jQuery Mail Form

You can use next fields in current version 1.0:

  • input type=text
  • textarea
  • input type=checkbox
  • input type=radio
  • select

class="Data-MailForm" - setup class to activate form
You can set next settings to the form:
data-mailform-subject="Collect Resume" - set mail subject
data-mailform-mailto="artur.collin@gmail.com" - set mail receiver
data-mailform-client-message="Thank you for your resume!" - set back mail for client

class="Data-MailForm-Button" - setup class to activate form
